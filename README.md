# SaaSify - Hyper-Modern SaaS Landing Page

![Project Banner](public/og.png)

A high-performance, visually stunning SaaS landing page template built with the latest web technologies. Featuring a "Void Black" aesthetic, deep fluid animations, and interactive 3D components.

## âœ¨ Key Features

- **Hero Section**: 3D Container Scroll animation revealing a high-fidelity dashboard mockup.
- **Floating Navbar**: A glassmorphic "island" navigation that persists smoothly on scroll.
- **Bento Grid**: Apple-style feature grid with mouse-following spotlight borders.
- **Velocity Scroll**: Physics-based horizontal scrolling for testimonials/logos.
- **Pricing**: Interactive cards with "Moving Border" effects and 3D meteors.
- **FAQ**: Minimalist accordion with background sparkle particles.
- **Performance**: Optimized for 60fps animations and instant load times.
